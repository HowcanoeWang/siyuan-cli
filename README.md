# 思源轻量版 (siyuan-cli)


在命令行中使用思源，用于离电(断网)使用时，减少耗电大户electron的能耗，提供更持久的续航。

需要在本地安装并运行完整的思源笔记(可只运行内核)，仅提供思源API支持的基本编辑功能，且使用过程需鼠标/触摸板支持(方便快速定位，不会有人想用方向键慢慢移动吧)。


## 支持特性

* [ ] 工作空间切换
* [ ] 文档树/书签
* [ ] 块编辑
* [ ] 图片简单预览
* [ ] 使用用户定义的快捷键

## 不支持的特性

* HTML渲染
* 布局调整
* 关系图
* 主题、插件、挂件等集市功能
* AI
* 更改设置
* 闪卡
* 导出与导入
* 云同步
